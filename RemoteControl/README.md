# 远程控制工具

通过手机浏览器远程控制电脑的轻量级工具。

## 功能特性

- 📱 手机浏览器直接访问，无需安装App
- 🖥️ 实时屏幕共享
- 🖱️ 触摸控制鼠标（点击、双击、长按右键、滚动）
- ⌨️ 键盘输入支持
- 🔒 可选密码保护

## 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 启动服务

```bash
python server.py
```

### 3. 手机访问

启动后会显示访问地址，例如：
```
手机访问: http://192.168.1.100:5000
```

在手机浏览器中打开此地址即可。

> ⚠️ 手机和电脑需要在同一WiFi网络下

## 操作说明

| 操作 | 效果 |
|------|------|
| 单指点击 | 鼠标左键点击 |
| 快速双击 | 鼠标双击 |
| 长按 (>0.5秒) | 鼠标右键点击 |
| 双指滑动 | 滚轮滚动 |
| 单指滑动 | 移动鼠标 |

### 底部快捷键

- **ESC**: 发送Escape键
- **Tab**: 发送Tab键
- **Enter**: 发送回车键
- **←**: 发送退格键

### 键盘输入

点击顶部"键盘"按钮，在输入框中输入文字，按回车发送。

## 配置说明

在 `server.py` 中修改 `CONFIG` 字典：

```python
CONFIG = {
    "host": "0.0.0.0",      # 监听地址
    "port": 5000,           # 服务端口
    "fps": 10,              # 帧率 (5-15 推荐)
    "quality": 60,          # JPEG质量 (50-80 推荐)
    "scale": 0.5,           # 缩放比例 (降低可减少带宽)
    "password": "",         # 访问密码 (空=不需要)
}
```

## 安全建议

1. 仅在信任的局域网中使用
2. 如需公网访问，请设置密码并使用反向代理+HTTPS
3. 使用完毕后关闭服务

## 技术栈

- **后端**: Flask + Flask-Sock (WebSocket)
- **屏幕捕获**: mss
- **鼠标键盘控制**: pyautogui
- **前端**: 原生HTML5 + Canvas + WebSocket
